# LLM Configuration

Summit AI Notes supports multiple Large Language Model (LLM) providers for generating meeting summaries and insights. You can choose between local processing for maximum privacy or cloud providers for advanced capabilities.

## Supported LLM Providers

### Local LLMs (Recommended for Privacy)

#### LM Studio
- **Privacy**: 100% local, no data leaves your device
- **Cost**: Free (one-time LM Studio setup)
- **Performance**: Depends on your Mac's specs
- **Best for**: Privacy-conscious consultants, confidential meetings

See the [LM Studio Setup Guide](/docs/setup/lmstudio) for detailed configuration instructions.

### Cloud LLM Providers

#### OpenAI (GPT-4, GPT-3.5)
- **Privacy**: Data sent to OpenAI servers
- **Cost**: Pay-per-use API pricing
- **Performance**: Excellent quality, fast
- **Best for**: Maximum quality summaries, complex analysis

#### Anthropic Claude
- **Privacy**: Data sent to Anthropic servers
- **Cost**: Pay-per-use API pricing
- **Performance**: Excellent for long-form content
- **Best for**: Detailed meeting analysis, comprehensive summaries

#### Google Gemini
- **Privacy**: Data sent to Google servers
- **Cost**: Pay-per-use API pricing (includes free tier)
- **Performance**: Good quality, competitive pricing
- **Best for**: Budget-conscious users, multilingual meetings

## How to Configure Your LLM Provider

### Step 1: Open Settings

1. Launch Summit AI Notes
2. Click on **Summit AI Notes** in the menu bar
3. Select **Settings** (or press `âŒ˜,`)
4. Navigate to the **LLM** tab

### Step 2: Choose Your Provider

Select your preferred provider from the dropdown:
- **LM Studio** (local)
- **OpenAI**
- **Anthropic Claude**
- **Google Gemini**

### Step 3: Enter API Credentials

#### For LM Studio (Local)
1. Ensure LM Studio is running
2. The default local endpoint is `http://localhost:1234/v1`
3. No API key required

#### For Cloud Providers
1. Obtain an API key from your chosen provider:
   - [OpenAI API Keys](https://platform.openai.com/api-keys)
   - [Anthropic Console](https://console.anthropic.com/)
   - [Google AI Studio](https://makersuite.google.com/app/apikey)
2. Paste your API key in the designated field
3. Click **Save Settings**

### Step 4: Test Your Configuration

1. Click the **Test Connection** button
2. Summit AI Notes will verify the connection and display the result
3. If successful, you're ready to generate summaries!

## Privacy Considerations

### Local LLMs (LM Studio)
- **Audio**: Never leaves your device
- **Transcriptions**: Processed locally
- **Summaries**: Generated locally
- **Result**: Complete privacy

### Cloud LLMs (OpenAI, Claude, Gemini)
- **Audio**: Never sent to cloud (only transcription text)
- **Transcriptions**: Sent to the LLM provider for summarization
- **Summaries**: Generated by cloud service
- **Result**: Transcription data shared with provider

> **Important**: When using cloud LLMs, only the transcription text is sent to the provider. Your audio recordings always stay on your device.

## Troubleshooting

### "Connection Failed" Error

**For LM Studio:**
1. Verify LM Studio is running
2. Check that the local server is started (port 1234)
3. Ensure no firewall is blocking localhost connections
4. Try restarting LM Studio

**For Cloud Providers:**
1. Verify your API key is correct (no extra spaces)
2. Check your API key has sufficient quota/credits
3. Ensure your internet connection is stable
4. Verify the API key has the correct permissions

### "Slow Summary Generation"

**For LM Studio:**
- Try a smaller model (7B instead of 70B)
- Check your Mac's CPU/GPU usage
- Ensure sufficient RAM is available
- Consider upgrading your hardware or using cloud LLMs

**For Cloud Providers:**
- This is rare; usually indicates network issues
- Check your internet connection speed
- Try switching to a "faster" model variant

### "API Quota Exceeded"

**For Cloud Providers:**
1. Check your account balance/credits
2. Review your API usage limits
3. Add payment method or purchase more credits
4. Consider switching to LM Studio for unlimited local use
